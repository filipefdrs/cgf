Computação Gráfica

Competências e Objetivos da Unidade Curricular

Apresentar e exercitar os conceitos básicos de computação gráfica 2D e 3D, implementando softwares queutilizam periféricos com capacidade gráfica.

Ementa/Conhecimentos

Fundamentos de computação gráfica. Percepção visual 2D e 3D. Representação vetorial e matricial de imagens. Transformações geométricas. Curvas e superfícies. Representação e modelagem. Animação. Realismo visual e iluminação. API de computação gráfica.

Conteúdo Programático
Introdução Computação Gráfica
  • Síntese, Processamento e Análise de Imagens
  • Imagens Digitais
  • Imagens Rasterizadas e Vetorizadas
  • Formação e Percepção de imagens 3D
  • Mercado de trabalho e conclusões
Introdução à API Canvas e Transformações
  • API de Computação Gráfica - Canvas
  • Representação Formal e Translação
  • Rotação
Introdução a API WebGL e Biblioteca Three.JS
